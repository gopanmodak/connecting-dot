# Conecting Dots Landing Page

A responsive, modern React landing page for connecting dots, built with **React**, **Tailwind CSS**, and **DaisyUI**.  
It features a hero section, feature cards, about section, statistics, and multiple informational sections. Includes **Terms & Conditions** and **Refund Policy** pages.
